# الكتابُ الحادي عشر: لُزومُ الجَماعةِ والاعتِصامُ بالكِتابِ والسُّنَّة وذَمُّ البِدَعِ ووُجوبُ قيامِ الإمامةِ

وفيه 4 كتب:

1. البابُ الأوَّلُ: الحَثُّ على لُزومِ الجَماعةِ وذَمُّ التفَرُّقِ
2. البابُ الثَّاني: الاعتِصامُ بالكِتابِ والسُّنَّة
3. البابُ الثَّالِثُ: ذَمُّ البِدَعِ
4. البابُ الرَّابِعُ: الإمامةُ
